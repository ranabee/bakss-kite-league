# BAKSS Kite League Manager

A production-grade kite sparring league management app built with:
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (Mobile-first PWA)
- **Backend**: Supabase PostgreSQL with Realtime
- **Hosting**: GitHub Pages

## Features
- 📊 Live leaderboards with instant updates
- ✂️ Record kite cuts with winner/loser/date/season
- 🏆 Champion tracking (day/month/year/season)
- 📈 Charts, reports, and statistics
- 🔄 Head-to-head rivalry matrix
- 📱 PWA - installable on Android & iPhone
- 🔐 Admin panel (PIN: 7860) for full management
- 💾 Export to JSON/CSV
- 🌐 Realtime sync across all devices

## Deployment
1. Push to GitHub (`main` branch)
2. Go to Settings → Pages → Source: GitHub Actions
3. The workflow deploys automatically

## Admin Features (PIN: 7860)
- Edit/delete players and matches
- Merge duplicate players
- Deactivate/reactivate players
- Reset seasons
- Export data backups
